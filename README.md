# Calculatrice Web Interactive

## Description du projet
Une calculatrice web simple, moderne et interactive développée en HTML, CSS et JavaScript. Ce projet permet de réaliser des opérations mathématiques basiques (addition, soustraction, multiplication, division) avec une interface utilisateur intuitive et responsive. La calculatrice dispose d'une gestion complète des erreurs et d'un support complet du clavier.

## Technologies utilisées
- **HTML5** : Structure et sémantique du document
- **CSS3** : Mise en forme, dégradé, animations et responsive design
- **JavaScript (Vanilla)** : Logique de la calculatrice et gestion des événements
- **Git** : Contrôle de version (optionnel)

## Instructions d'exécution
1. Téléchargez ou clonez le projet
2. Ouvrez le fichier `index.html` dans un navigateur web moderne (Chrome, Firefox, Safari, Edge)
3. Utilisez les boutons souris ou votre clavier pour effectuer des calculs
4. Appuyez sur le bouton "=" ou la touche Entrée pour obtenir le résultat

### Raccourcis clavier supportés
- **Chiffres** (0-9) : entrer les nombres
- **Opérateurs** (+, -, *, /) : effectuer des opérations
- **Entrée** : calculer le résultat
- **Retour arrière** : supprimer le dernier chiffre
- **Échap** : réinitialiser la calculatrice
- **Virgule/Point** : ajouter une décimale

## Captures d'écran ou résultats
- Interface modernes avec un design en dégradé violet/rose
- Affichage en temps réel des calculs
- Messages d'erreur clairs pour les opérations invalides
- Boutons avec feedback visuel (hover, active, shadow)
- Adaptation automatique à tous les appareils (desktop, tablette, mobile)

### Exemple de fonctionnement
```
Entrée : 15 + 25
Résultat : 40

Entrée : 100 / 5
Résultat : 20

Entrée : 10 / 0
Résultat : Erreur: Division par zéro
```
